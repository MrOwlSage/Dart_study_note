## 级联
对统一对象进行一系列操作是，用级联"\.\."
```Dart
class Person {
  String name;
  String country;

  void setCountry(String tCountry) {
    country = tCountry;
  }

  // 默认的
  String toString() => "****Name: $name\nCountry:$country****";
}

int main {
Person p = Person() ..name = "123"
                      ..setCountry("China");
p ..name = "zyr"
  ..setCountry('Island');

// print(p) === print(p.toString())
print(p); 
  
}

```
